Here's a list of emerging AI startups and open-source communities developing agentic frameworks:

**Open-Source Agentic Frameworks and Associated Entities:**

*   **CrewAI:** An open-source framework for orchestrating multiple specialized agents in collaborative workflows. A startup named CrewAI is leveraging this for customer service and support solutions.
*   **LangChain:** A widely adopted open-source framework for building language model applications, offering modular components for creating sophisticated agentic workflows. Developed by the company LangChain and used by organizations like Klarna, Elastic, and Rakuten.
*   **MetaGPT:** An open-source agentic AI framework known for its multi-agent collaboration capabilities, particularly in software development and business process automation.
*   **XAgent:** An open-source framework focusing on autonomous planning and execution through a hierarchical task planning framework.
*   **Auto-GPT:** Enables autonomous task execution using GPT models.
*   **AgentGPT:** Provides a browser-based interface for creating and deploying Auto-GPT-style agents.
*   **SuperAGI:** A production-ready framework emphasizing extensibility, with a graphical interface and support for various tools and APIs.
*   **AutoGen by Microsoft:** Facilitates structured interactions and collaborative chat logic among agents.
*   **LangGraph:** Utilizes a graph-based approach for building and connecting agents.
*   **SmolAgents:** Another popular open-source framework for building AI agents.
*   **Atomic Agents:** Focuses on schema-driven building blocks for structured and controlled agent development.
*   **Mastra:** Developed by the team behind Gatsby, this is a JavaScript framework tailored for frontend developers to integrate agents into their ecosystems.
*   **Adopt AI:** Offers a tooling layer that assists in connecting application APIs with callable tools for agent execution, working in conjunction with existing open-source frameworks.
