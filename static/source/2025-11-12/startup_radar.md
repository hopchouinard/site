Here's a summary of emerging startups and open-source communities developing notable AI applications, particularly agentic frameworks:

**Open-Source Agentic Frameworks:**

*   **LangChain:** A popular framework for building LLM-powered applications, offering modular components for various AI tasks.
*   **LangGraph:** Built on LangChain, it enables graph-based workflows for complex agent systems with fine-grained orchestration.
*   **AutoGen (Microsoft):** A multi-agent framework designed for scalable agent systems, inter-agent communication, and orchestration.
*   **CrewAI:** Specializes in multi-agent collaboration, allowing AI agents to work together through defined roles and shared goals.
*   **AutoGPT:** Focuses on autonomous task execution using large language models.
*   **Semantic Kernel (Microsoft):** An enterprise-ready framework for connecting LLMs with real-world tools, data, and APIs.
*   **SmolAgents:** A lightweight and educational framework for agentic AI.
*   **MetaGPT:** Simulates a software development team structure to automate workflows.
*   **Atomic Agents:** An emerging framework for decentralized, multi-agent systems, focusing on scalability.
*   **kagent:** An open-source framework for running AI agents in Kubernetes, aimed at DevOps and platform engineers.
